# Seedr Telegram Bot

A Telegram bot to manage your Seedr.cc account. Add torrents, list files, and download with ease.

## Features

✅ Add magnet links  
✅ Get download links  
✅ Delete files  
✅ List files  

## Deployment

1. Clone the repo  
2. Install requirements  
3. Create `.env` file  
4. Run `python main.py`
